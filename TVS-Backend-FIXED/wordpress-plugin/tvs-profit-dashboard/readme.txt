Connects WooCommerce to TVS Profit Dashboard for real-time profit analytics.

== Description ==
This plugin connects your WooCommerce store to the TVS Profit Dashboard backend:
- Add COGS (Cost of Goods Sold) to each product
- Bulk COGS editor for all products
- Auto-sends order webhooks to backend
- Shows margin % in product list
- Dashboard widget with quick links
- Admin bar quick access

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/
2. Activate in WordPress Admin > Plugins
3. Go to TVS Dashboard > Settings
4. Enter your backend API URL and shared secret
5. Go to TVS Dashboard > COGS Manager to set product costs
